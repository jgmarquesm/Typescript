• Criar um html;
  • Adcionar o caminho do script JS ao HTML;
  • Na pasta raiz, abra o terminal e rode 'npm init -y'
  • Em seguida, rode: 'npm i -s live-server'
  • Abra o arquivo 'package.json' e mude configuração '"scripts"' para: ' "scripts": { "start": "live-server" } '
  • No terminal rode 'tsc -w'.
  • Para rodar o server, no terminal digite 'npm start'.
